# EK-RAG PPT Color Palette (Octave Showcase Brand)

Use this palette for PowerPoint slides, speaker decks, and infographic extensions so they match the Octave showcase deck (`ek-rag-showcase.html`): black-dominant background, white text, gray hierarchy, and neon accent highlights.

## Base (black-dominant)
- Background: `#1A1A1F`
- Panel: `#23232A`
- Panel raised: `#2C2C34`
- Border / hairline: `#3E4047`

## Text hierarchy
- Primary text: `#FFFFFF`
- Lead text: `#CBD0D8`
- Body text: `#B2B8C4`
- Muted / micro text: `#6F7480`

## Accent highlights (one accent per section, showcase rhythm)
- Aqua: `#00FFFF`
- Blue: `#0000FF`
- Green: `#4FFF00`
- Pink: `#FF00C7`
- Yellow: `#FFF500`
- Orange: `#F46600`

## Section accent mapping in the infographic
- 01 Problem Statement: Orange `#F46600`
- 02 Product Overview (hero): Aqua `#00FFFF`
- 03 Input Sources: Aqua `#00FFFF`
- 04 Data Processing Pipeline: Blue `#0000FF`
- 05 AI Engine: Pink `#FF00C7`
- 06 User Experience: Aqua `#00FFFF`
- 07 Enterprise Features: Yellow `#FFF500`
- 08 Technology Stack: Green `#4FFF00`
- 09 Key Benefits: Green `#4FFF00`
- 10 End Result (hero): Aqua `#00FFFF`

## PowerPoint theme mapping
- Background 1: `#1A1A1F`
- Text 1: `#FFFFFF`
- Text 2: `#B2B8C4`
- Accent 1: `#00FFFF`
- Accent 2: `#0000FF`
- Accent 3: `#4FFF00`
- Accent 4: `#FF00C7`
- Accent 5: `#F46600`
- Accent 6: `#FFF500`

## Hero gradient
- `linear-gradient(135deg, #00FFFF 0%, #0000FF 55%, #FF00C7 100%)`

## Usage rules (from the brand system)
- Keep the background black-dominant; use accents only for eyebrows, section numbers, one highlighted keyword, thin rules, arrows, and data-viz.
- Do not tint large surfaces with accents; panels stay `#23232A` / `#2C2C34`.
