# =============================================================================
# EK-RAG MASTER REQUIREMENTS (all services + tests)
# -----------------------------------------------------------------------------
# Single aggregated dependency set to run and TEST every Python service:
#   EKIE (ingestion), EKRE (retrieval), EKCP (chat platform), EKDC (doc convert),
#   and the shared contracts package.
#
# Install into one virtual environment:
#     python -m venv .venv
#     .\.venv\Scripts\Activate.ps1        # Windows PowerShell
#     pip install -r requirements-all.txt
#
# GPU NOTE (NVIDIA): `torch` from PyPI installs the CPU-only build. For GPU
# acceleration install a CUDA build that matches your driver INSTEAD, e.g.:
#     pip install torch==2.12.1 --index-url https://download.pytorch.org/whl/cu130
#   (use cu128/cu126 for older drivers), then set, per service .env:
#     *_EMBEDDING__DEVICE=auto  and  *_EMBEDDING__TORCH_DTYPE=float16
#
# NOTE: EKDC (media -> Markdown) is a heavy, optional stack and normally runs in
# its OWN venv (services/ekdc/.venv). It is included here so a single env can
# test everything; comment out the "EKDC" block if you do not need it.
# The web UI (apps/web-ui) is Node.js/Next.js and is NOT covered by this file.
# =============================================================================
