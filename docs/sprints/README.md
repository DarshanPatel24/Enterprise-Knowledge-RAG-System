# Sprint Track Catalog

This folder contains execution-ready sprint tracks.

## Naming Standard
1. Use lowercase kebab-case.
2. Use `-sprint-track.md` suffix for all sprint track files.
3. Keep one file per track to simplify ownership and gate reviews.

1. Track Definition Of Ready
2. Track Definition Of Done
3. Success Metrics (Track Level)
4. Sprint blocks with objective, scope, out-of-scope, stories, deliverables, acceptance, exit evidence
